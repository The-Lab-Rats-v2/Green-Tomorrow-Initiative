# Green Tomorrow Initiative

Green Tomorrow Initiative is a responsive static website for an environmental non-profit focused on sustainability, community education, and local action.

The site presents the organization, its programmes, latest updates, and contact information through a clean HTML, CSS, and JavaScript codebase.

## Pages

- `index.html` - Homepage with hero, about preview, featured programmes, news preview, and contact call-to-action
- `about.html` - About page
- `programmes.html` - Programmes page
- `news.html` - News page
- `contact.html` - Contact page

## Running the Website

This is a static website and does not require a build step.

Open `index.html` in a browser to view the site locally. The homepage uses remote image URLs, so an internet connection is required for those images to load.

## Notes

- Shared navigation and footer content appear across all pages.
- Mobile navigation is handled by `assets/js/navigation.js`.
- Global styling is in `assets/css/style.css`.
- Responsive layout rules are in `assets/css/responsive.css`.
